* Docker commands
** run - start a container
#+begin_src shell
docker run nginx
#+end_src

** ps - list container
#+begin_src shell

#+end_src

* Docker Run
** Run - Tag
#+begin_src shell
docker run redis
#+end_src

#+begin_src shell
docker run redis:4.0
#+end_src

#+begin_src shell
docker run kodekloud/simple-prompt-docker
#+end_src

#+RESULTS:
| Welcome! | Please | enter   | your | name: |
| Hello    | and    | Welcome | !    |       |

** Run - PORT mapping
#+begin_src
docker run -p 80:5000 kodekloud/webapp
#+end_src

#+begin_src shell
docker run -p 3306:3306 mysql
#+end_src

** Run - Volume mapping

#+begin_src shell
docker run mysql
#+end_src

#+begin_src shell
docker stop mysql
docker rm mysql
#+end_src

#+begin_src shell
docker run -v /opt/daradir:/var/lib/mysql mysql
#+end_src

** Inspect container
#+begin_src shell
docker inspect nostalgic_galois
#+end_src

** Container logs
#+begin_src shel
docker logs nostalgic_galois
#+end_src
